常见疑问

MinGW和MSVC选择哪一个？

首先明确一点，如果只是使用QtCreator作为IDE，并没有使用Qt中的库那么完全不用考虑这个问题好像不是，还是需要使用里面带的Win32API。

MSVC：MicroSoft Visual Studio

MinGW：Minimalist GNU for Windows ，又称mingw32，是将GCC编译器和GNU Binutils移植到Win32平台下的产物，包括一系列头文件（Win32API）、库和可执行文件。另有可用于产生32位及64位Windows可执行文件的MinGW-w64项目，是从原本MinGW产生的分支。
